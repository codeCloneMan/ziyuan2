UI Test scaffold for navigation and theme features

- Prerequisites: Node.js, Playwright
- Install: npm i -D @playwright/test
- Run tests: npx playwright test
- This repo includes a sample test at tests/e2e/navigation.spec.ts
- Extend tests to cover:
  - Mobile navigation open/close
  - Drag/slide gestures for mobile nav
  - Language switch and fashion/theme toggles
