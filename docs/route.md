# 学习路线

欢迎使用字源形码！按照以下步骤，你将轻松掌握这款输入法。

## 📚 第一阶段：了解字源形码

### 1.1 阅读入门介绍
- [快速入门](/introduction) - 了解字源形码的基本概念和优势
- [字源了解](/xingma/liaojie) - 深入了解字源形码的设计理念

### 1.2 掌握核心概念
- 字源形码基于 20 个核心字根
- 四码定长，最多输入 4 个字母
- 遵循汉字自然拆分规律

## 🎯 第二阶段：学习字根

### 2.1 顺序练习（推荐）
- [顺序练习](/practice/modern) - 按顺序逐个学习所有字根
- 系统化记忆，不易遗漏
- 每个字根都有提示信息

### 2.2 随机练习
- [随机练习](/practice/random) - 随机抽测字根
- 检验真实掌握程度
- 巩固记忆效果

### 2.3 重点突破
- [常用字根练习](/practice/top500) - 专项练习高频字根
- 快速提升日常打字速度
- 覆盖 90% 以上常用汉字

## ⚡ 第三阶段：实战训练

### 3.1 综合练习
- [所有字根练习](/practice/typing) - 完整的字根练习
- 模拟真实打字场景
- 提升整体打字速度

### 3.2 错题复习
- [错误字根练习](/practice/error) - 专门练习错误字根
- 自动记录错误字根
- 针对性强化薄弱环节

## 🚀 第四阶段：进阶提升

### 4.1 速度训练
- 使用 52打字 等工具进行速度测试
- 目标：每分钟 60 字以上
- 持续练习，不断突破

### 4.2 词库扩展
- 学习更多词组输入
- 提升长句输入效率
- 减少编码冲突

### 4.3 实际应用
- 日常打字优先使用字源形码
- 在工作、学习中实践
- 形成肌肉记忆

## 💡 学习建议

### 时间安排
- **新手期（1-3天）**：每天 30-60 分钟，完成字根学习
- **练习期（1-2周）**：每天 20-30 分钟，巩固字根记忆
- **进阶期（持续）**：在日常使用中不断熟练

### 学习技巧
1. **理解记忆**：不要死记硬背，理解字根与汉字的关系
2. **多次重复**：短时多次比长时间一次更有效
3. **实际应用**：尽快在真实场景中使用
4. **错题巩固**：重点练习错误字根

### 常见问题
**Q：多久能学会？**
A：一般 30 分钟可以上手掌握 20 个字根，1-2 周内可以熟练使用。

**Q：和五笔相比有什么优势？**
A：字根更少（20个 vs 130+），学习曲线更平缓，上手更快。

**Q：打字速度能达到多少？**
A：经过 1-2 周练习，大多数用户可以达到每分钟 60 字以上。
