## 使用說明

### 🔍 快速搜尋
在搜尋框輸入字根字元、提示文字、編碼或分類，即可快速找到目標字根。

### 🏷️ 分類篩選
點擊分類標籤可查看特定類別字根：
- **全部**：顯示所有字根
- **自然**：與自然相關的字根
- **人體**：與人體相關的字根
- **動物**：與動物相關的字根
- **植物**：與植物相關的字根
- **數字**：數字相關字根
- **筆畫**：基本筆畫字根
- **結構**：字形結構部件
- **其他**：其他字根

### 🔤 編碼篩選
透過下拉選單選擇特定編碼（A-Z），查看該編碼對應字根。

### 📊 排序方式
- **按編碼**：按字母順序
- **按字元**：按 Unicode 順序

### 📄 分頁瀏覽
每頁顯示 50 個字根，可透過分頁按鈕查看更多內容。

## 學習建議

1. 先掌握 20 個核心字根
2. 按分類分批記憶
3. 建立編碼與字根聯想
4. 搭配練習頁反覆鞏固
