## 練習說明

錯題本專門收集您在練習中出錯的字根，進行針對性強化訓練。

## 練習特點

- **錯題收集**：自動收集練習中的錯誤字根
- **針對性訓練**：專門練習容易出錯的字根
- **智能排序**：按錯誤頻率優先練習
- **錯題清理**：掌握後可清除錯題記錄

## 統計信息

- **錯題總數**：顯示當前錯題本中的字根數量
- **錯誤頻率**：每個字根的錯誤次數統計
- **掌握程度**：根據正確率評估掌握程度
- **進度追蹤**：錯題掌握進度可視化

## 練習模式

- **錯題模式**：只練習錯題本中的字根
- **強化模式**：對高錯誤率字根重點練習
- **複習模式**：定期複習已掌握的錯題

## 快捷鍵

| 按鍵 | 功能 |
|------|------|
| Enter | 提交/繼續 |
| Ctrl+→ | 跳過當前字根 |
| Esc | 暫停/繼續 |
| Delete | 從錯題本中移除當前字根 |
