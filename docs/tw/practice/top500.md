## 練習說明

常用字根練習專門針對最高頻的500個字根進行重點訓練，幫助您快速提升打字速度。

## 練習特點

- **高頻專項**：專注於最常用的500個字根
- **效率提升**：快速提升日常打字效率
- **分組練習**：按使用頻率分組練習
- **統計追蹤**：詳細的練習統計數據

## 練習模式

- **順序模式**：按頻率順序逐個練習
- **隨機模式**：隨機抽測常用字根
- **錯題模式**：專門練習錯誤的字根

## 快捷鍵

| 按鍵 | 功能 |
|------|------|
| Enter | 提交/繼續 |
| Ctrl+→ | 跳過當前字根 |
| Esc | 暫停/繼續 |
