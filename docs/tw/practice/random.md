## 練習說明

隨機練習會打亂字根順序，幫助您鞏固記憶，檢驗真實掌握程度。

## 練習特點

- **隨機抽測**：從所有字根中隨機選擇進行練習
- **記憶檢驗**：真實檢驗對字根的掌握程度
- **即時反饋**：正確/錯誤即時提示
- **進度保存**：練習進度自動保存到本地

## 快捷鍵

| 按鍵 | 功能 |
|------|------|
| Enter | 提交/繼續 |
| Ctrl+→ | 跳過當前字根 |
| Esc | 暫停/繼續 |
