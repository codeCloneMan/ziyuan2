## 練習說明

順序練習按照字根表的順序，系統化學習所有字根的編碼。

## 練習特點

- **系統學習**：按順序逐個學習字根編碼
- **分組練習**：每組8個字根，每個字根練習4遍
- **即時反饋**：正確/錯誤即時提示
- **進度保存**：練習進度自動保存到本地

## 快捷鍵

| 按鍵 | 功能 |
|------|------|
| Enter | 提交/繼續 |
| Ctrl+→ | 跳過當前字根 |
| Esc | 暫停/繼續 |
