# 短拼拼音

短拼拼音是字源形碼配套的拼音輸入方案，專為提升拼音輸入效率而設計。

## 🎯 設計理念

短拼拼音通過簡化拼音輸入，結合智能預測技術，大幅提升拼音輸入的效率和準確性。

### 核心特點

- **簡化輸入**：縮短拼音輸入長度
- **智能預測**：基於語境智能預測
- **高效選詞**：快速選擇目標詞語
- **學習適應**：根據用戶習慣自適應

## 🔤 輸入規則

### 基本規則

1. **聲母簡化**：常用聲母簡化輸入
2. **韻母簡化**：常用韻母縮短輸入
3. **聲調處理**：可選聲調輸入
4. **詞組預測**：智能詞組預測

### 簡化方案

#### 聲母簡化
- **zh → z**：如 "zhang" → "zang"
- **ch → c**：如 "chang" → "cang"
- **sh → s**：如 "shang" → "sang"

#### 韻母簡化
- **ang → ag**：如 "fang" → "fag"
- **eng → eg**：如 "feng" → "feg"
- **ing → ig**：如 "fing" → "fig"
- **ong → og**：如 "fong" → "fog"

#### 特殊簡化
- **雙字母簡化**：部分雙字母簡化為單字母
- **常見詞組**：常用詞組特殊簡化
- **用戶自定義**：支持用戶自定義簡化規則

## 📖 使用方法

### 基本操作

1. **啟用短拼**：在輸入法設置中啟用短拼模式
2. **簡化輸入**：使用簡化規則輸入拼音
3. **選擇詞語**：從候選詞中選擇目標詞語
4. **確認輸入**：確認輸入或繼續編輯

### 進階技巧
