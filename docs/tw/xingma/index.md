# 字源形碼

字源形碼是一款入門簡單且性能強大的形碼輸入法，專為現代用戶設計。

## 🎯 設計理念

字源形碼基於漢字字源和結構規律，採用科學的拆分方法，讓輸入更加自然高效。

### 核心特點

- **簡單易學**：僅需20個核心字根，30分鐘即可上手
- **科學拆字**：基於漢字自然結構，符合直覺
- **高效輸入**：四碼定長，低重碼率，打字流暢
- **大字集支持**：支持10萬漢字，覆蓋日常使用需求

## 🔤 字根系統

### 字根分佈

字源形碼的20個核心字根科學地分佈在26個鍵位上：

```
Q W E R T Y U I O P
 A S D F G H J K L
  Z X C V B N M
```

### 字根分類

#### 基礎字根
- **橫豅點撇**：基本筆畫字根
- **偏旁部首**：常見偏旁部首
- **獨體字**：常用獨體字

#### 組合字根
- **形近字根**：字形相似的字根
- **音近字根**：讀音相近的字根
- **義近字根**：意義相關的字根

## 📖 編碼規則

### 基本原則

1. **取大優先**：優先取最大字根
2. **順序優先**：按書寫順序取碼
3. **結構優先**：考慮漢字結構特點
4. **頻率優先**：常用字優先考慮

### 編碼長度

- **一碼字**：超高頻常用字
- **二碼字**：高頻常用字
- **三碼字**：一般常用字
- **四碼字**：複雜字和低頻字

### 特殊規則

- **簡碼處理**：常用字簡化編碼
- **重碼處理**：智能重碼選擇
- **容錯處理**：允許一定容錯範圍

## 🎮 學習方法

### 初學階段

1. **熟悉字根**：掌握20個核心字根
2. **理解規則**：理解基本編碼規則
3. **順序練習**：按順序練習所有字根
4. **鞏固記憶**：通過重複練習鞏固

### 進階階段

1. **隨機練習**：隨機抽測字根記憶
2. **常用字根**：重點練習高頻字根
3. **錯題複習**：專門練習錯誤字根
4. **實戰應用**：在日常使用中練習

### 熟練階段

1. **速度提升**：提升打字速度和準確性
2. **盲打訓練**：實現不看鍵盤輸入
3. **詞組輸入**：學習詞組和短語輸入
4. **高級技巧**：掌握各種輸入技巧

## 📊 性能指標

### 輸入效率

- **平均碼長**：2.8碼/字
- **重碼率**：低於1%
- **首選命中率**：高於95%
- **輸入速度**：可達100字/分鐘

### 學習效率

- **入門時間**：30分鐘掌握字根

- **精通時間**：1個月達到高效水平
- **記憶難度**：相比五筆降低70%

## 🚀 發展歷程

### 誕生背景

字源形碼誕生於現代輸入法發展的關鍵時期，旨在解決傳統形碼學習難度大、拼音輸入效率低的問題。

### 發展階段

1. **研究階段**：深入研究漢字結構和輸入法理論
2. **設計階段**：科學設計字根系統和編碼規則
3. **開發階段**：開發輸入法引擎和用戶界面
4. **優化階段**：根據用戶反饋持續優化改進
