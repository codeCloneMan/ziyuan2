## 发现的问题与建议

### 1. RootPractice.vue 组件 (docs/.vitepress/components/RootPractice.vue)

**问题：**
- ❌ 组件过于庞大（800+ 行），违反单一职责原则
- ❌ 状态管理复杂，多个相关状态分散
- ❌ 重复代码较多（如进度保存逻辑）
- ❌ 缺少 TypeScript 类型定义
- ❌ 部分魔法数字和硬编码字符串

**建议改进：**
```javascript
// 1. 提取状态管理到 Pinia 或自定义 hook
const usePracticeState = () => {
  const currentRoot = ref(null)
  const userInput = ref('')
  // ... 集中管理所有状态
  return { currentRoot, userInput, ... }
}

// 2. 提取进度管理逻辑
const useProgressManagement = () => {
  const save = (data) => { /* ... */ }
  const load = () => { /* ... */ }
  const clear = () => { /* ... */ }
  return { save, load, clear }
}

// 3. 使用常量替代魔法数字
const CROSS_PRACTICE_GROUP_SIZE = 10
const CROSS_PRACTICE_REPETITIONS = 3
const FEEDBACK_DISPLAY_TIME = 1500

// 4. 添加 TypeScript 类型定义
interface ProgressData {
  mode: 'order' | 'shuffle'
  correctCount: number
  answeredRoots: number
  practiceRoots: Root[]
  isComplete: boolean
  timestamp: string
  version: string
}
```

### 3. rootData.js 数据文件 (docs/.vitepress/data/rootData.js)

**问题：**
- ⚠️ 数据量大（400+ 字根），但缺少分类和索引
- ⚠️ 部分字根编码重复（如 '见' 出现两次）
- ⚠️ 缺少字根的完整信息（如笔顺、示例等）

**建议改进：**
```javascript
// 1. 添加分类和索引
export const allRoots = [
  { character: '白', code: 'k', hint: '白色', category: '自然' },
  // ...
]

export const rootsByCode = new Map()
export const rootsByCategory = new Map()

// 初始化索引
allRoots.forEach(root => {
  rootsByCode.set(root.code, root)
  if (!rootsByCategory.has(root.category)) {
    rootsByCategory.set(root.category, [])
  }
  rootsByCategory.get(root.category).push(root)
})

// 2. 导出查询函数
export const findRootByCode = (code) => rootsByCode.get(code)
export const findRootsByCategory = (category) => rootsByCategory.get(category) || []
```

### 4. ImageZoom.vue 组件

**问题：**
- ⚠️ 缺少图片加载状态处理
- ⚠️ 缺少错误图片显示
- ⚠️ 没有加载失败的降级方案

**建议改进：**
```vue
<script setup>
const imageLoaded = ref(false)
const imageError = ref(false)

const handleImageLoad = () => {
  imageLoaded.value = true
}

const handleImageError = () => {
  imageError.value = true
  console.error('图片加载失败:', props.src)
}
</script>

<template>
  <img
    :src="imageError ? '/placeholder.png' : src"
    @load="handleImageLoad"
    @error="handleImageError"
    v-show="!imageError || imageLoaded"
  />
  <div v-if="imageError" class="error-placeholder">
    图片加载失败
  </div>
</template>
```

const CACHE_VERSION = '1.0'
const getCachedRoots = () => {

}
