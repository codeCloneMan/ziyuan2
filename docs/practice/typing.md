# 字源字根打字

<TypingPractice />

## 使用说明

这是一个仿照在线打字测试设计的字根练习页面，支持实时打字练习和速度测试。

### 功能特点

- 🎯 **实时打字** - 显示字根，输入对应编码
- ⚡ **速度统计** - 实时计算打字速度（字符/分钟）
- 📊 **准确率统计** - 统计正确率和错误数
- 🔄 **无限练习** - 可以持续练习，不断提升
- 🎨 **现代化界面** - 清晰的视觉设计

### 操作指南

1. 看到显示的字根
2. 使用键盘输入对应的编码
3. 自动跳转到下一个字根
4. 查看实时统计数据
5. 按ESC暂停/继续练习

### 统计信息

- **打字速度**: 每分钟输入的字根数量
- **正确率**: 正确输入的字根比例
- **已练习**: 总共练习的字根数量
- **用时**: 练习的总时间

### 快捷键

- `Enter` - 提交输入 / 跳过
- `ESC` - 暂停 / 继续练习
- `Tab` - 跳过当前字根
