# 字根练习

<ModernPractice mode="order" />

## 功能特点

### 🎯 智能练习
- **顺序模式**: 按字根顺序练习
- **随机模式**: 随机打乱练习顺序
- **自动跳转**: 答对自动进入下一个
- **手动控制**: 支持手动上一个/下一个

### 📊 实时统计
- **进度追踪**: 实时显示练习进度
- **正确率**: 自动计算正确率百分比
- **用时统计**: 精确计时功能
- **详细数据**: 正确/错误数量统计

### ⌨️ 键盘快捷键
- `Enter` - 提交答案或下一个
- `Tab` - 跳过当前字根
- `Ctrl+→` - 下一个字根
- `Ctrl+←` - 上一个字根
- `ESC` - 暂停/继续

### 🎨 现代化设计
- 渐变色主题
- 平滑动画效果
- 响应式布局
- 移动端适配

## 使用说明

1. **开始练习**: 页面自动开始练习
2. **输入编码**: 在输入框中输入字根对应的编码
3. **即时反馈**: 系统会立即判断对错
4. **查看统计**: 顶部实时显示练习数据
5. **完成练习**: 完成所有字根后显示总结

## 练习模式

### 顺序模式
按字根在数据中的顺序依次练习，适合系统性学习。

### 随机模式
随机打乱字根顺序练习，适合复习和巩固。

## 统计说明

- **正确率**: 正确答案占总答题数的百分比
- **进度**: 当前完成字根数/总字根数
- **用时**: 从开始到当前的累计时间
