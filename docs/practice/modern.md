## 练习说明

顺序练习按照字根表的顺序，系统化学习所有字根的编码。

## 练习特点

- **系统学习**：按顺序逐个学习字根编码
- **分组练习**：每组8个字根，每个字根练习4遍
- **即时反馈**：正确/错误即时提示
- **进度保存**：练习进度自动保存到本地

## 快捷键

| 按键 | 功能 |
|------|------|
| Enter | 提交/继续 |
| Ctrl+→ | 跳过当前字根 |
| Esc | 暂停/继续 |
