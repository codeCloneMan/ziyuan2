## 练习说明

随机练习打乱字根顺序，检验真实掌握程度，避免死记硬背顺序。

## 练习特点

- **随机顺序**：每次练习顺序不同
- **分组练习**：每组8个字根，每个字根练习4遍
- **即时反馈**：正确/错误即时提示
- **进度保存**：练习进度自动保存到本地

## 快捷键

| 按键 | 功能 |
|------|------|
| Enter | 提交/继续 |
| Ctrl+→ | 跳过当前字根 |
| Esc | 暂停/继续 |
