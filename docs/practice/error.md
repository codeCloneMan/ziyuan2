# 错误字根练习

<ErrorRadicalPractice />

## 功能特点

### 🎯 针对性练习
- **错误聚焦**: 只练习之前答错的字根
- **智能复习**: 系统自动记录错误字根
- **强化记忆**: 针对薄弱环节进行强化练习
- **进度跟踪**: 实时显示错误字根掌握情况

### 📊 智能统计
- **错误统计**: 记录每个字根的出错次数
- **掌握程度**: 显示字根的掌握进度
- **练习历史**: 查看历史错误记录
- **进步追踪**: 对比历史正确率变化

### 🎨 个性化设置
- **练习强度**: 可调整错误字根的练习次数
- **间隔复习**: 支持间隔重复记忆算法
- **难度调整**: 根据掌握程度自动调整难度
- **目标设定**: 可设置每日练习目标

## 使用说明

1. **开始练习**: 系统会自动加载您之前练习中答错的字根
2. **强化记忆**: 针对错误字根进行重点练习
3. **掌握检测**: 当字根连续答对3次后，系统会标记为已掌握
4. **进度查看**: 实时查看错误字根的掌握情况
5. **完成练习**: 当所有错误字根都掌握后，练习完成

## 练习原理

### 间隔重复记忆
基于科学的间隔重复记忆算法，在最佳时间点复习字根，提高记忆效率。

### 错误分析
系统会分析您的错误模式，找出容易混淆的字根组合，进行对比练习。

### 渐进式掌握
每个错误字根需要连续答对3次才能标记为已掌握，确保真正掌握。

## 统计说明

- **错误字根数**: 当前需要练习的错误字根数量
- **已掌握数**: 已经掌握的错误字根数量
- **掌握率**: 已掌握字根占总错误字根的百分比
- **平均出错次数**: 每个错误字根的平均出错次数
