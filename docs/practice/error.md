## 练习说明

错误字根练习专门针对练习过程中答错的字根，帮助巩固薄弱环节。

## 练习特点

- **智能收集**：自动记录练习中答错的字根
- **分组强化**：每组8个字根，每个字根练习4遍后移除
- **掌握移除**：连续答对指定次数后自动从错题本移除
- **进度保存**：练习进度自动保存到本地

## 快捷键

| 按键 | 功能 |
|------|------|
| Enter | 提交/继续 |
| Ctrl+→ | 跳过当前字根 |
| Esc | 暂停/继续 |

## 使用建议

1. 先进行[顺序练习](/practice/modern)或[随机练习](/practice/random)
2. 答错的字根会自动加入错题本
3. 定期练习错题本巩固薄弱字根
4. 掌握后字根会自动移除
