## 练习说明

常见根练习针对高频使用的字根，帮助快速提升打字效率。

## 练习特点

- **精选字根**：基于常用字的实际使用频率
- **分组练习**：每组8个字根，每个字根练习4遍
- **顺序/随机**：支持两种练习模式切换
- **进度保存**：练习进度自动保存到本地

## 快捷键

| 按键 | 功能 |
|------|------|
| Enter | 提交/继续 |
| Ctrl+→ | 跳过当前字根 |
| Esc | 暂停/继续 |

## 学习建议

1. 每天练习15-30分钟
2. 重点关注错误率高的字根
3. 在实际打字中应用所学
