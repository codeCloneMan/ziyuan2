# 扩展汉字字根显示说明

## 问题说明

某些字根（如"𠂊"）属于**CJK 扩展 B 区**字符（Unicode: U+20000-U+2A6DF），这些字符需要特殊字体支持才能正确显示。

## 字符信息

- **字符**：𠂊
- **Unicode**: U+2008A
- **所属区域**: CJK 扩展 B 区
- **读音**: 可能是"撇"的变体
- **用途**: 汉字部件，常见于"危"等字的顶部

## 支持的扩展区域

本项目已支持以下 CJK 扩展区域：

✅ **CJK 扩展 B 区** (U+20000-U+2A6DF) - 42,711 字
✅ **CJK 扩展 C 区** (U+2A700-U+2B73F) - 3,158 字
✅ **CJK 扩展 D 区** (U+2B740-U+2B81F) - 222 字
✅ **CJK 扩展 E 区** (U+2B820-U+2CEAF) - 5,762 字
✅ **CJK 扩展 F 区** (U+2CEB0-U+2EBEF) - 7,473 字
✅ **CJK 扩展 G 区** (U+30000-U+3134F) - 4,939 字
✅ **CJK 扩展 H 区** (U+31350-U+323AF) - 4,192 字
✅ **CJK 扩展 I 区** (U+323B0-U+32FFF) - 正在添加

## 解决方案

### 方案一：安装支持扩展汉字的字体（推荐）

#### Windows 系统

1. **下载字体**
   - [BabelStone Han](https://www.babelstone.co.uk/Software/Download.html) - **最全面**，支持扩展 B/C/D/E/F/G/H
   - [花园明朝体 (Hanazono)](http://fonts.jp/hanazono/) - 免费开源，支持扩展 B/C/D/E/F
   - [IPA 明朝](https://moji.or.jp/ipafont/) - 日本 IPA 字体
   - **Windows 10/11 自带**：SimSun-ExtB (宋体扩展 B)

2. **安装步骤**
   - 下载字体文件（.ttf 或 .otf）
   - 右键点击字体文件 → "为所有用户安装"
   - 重启浏览器

#### macOS 系统

macOS 默认支持大部分扩展汉字，如无法显示：
- 下载 [Source Han Sans](https://github.com/adobe-fonts/source-han-sans)
- 或 [Noto Sans CJK](https://fonts.google.com/noto)

#### Linux 系统

```bash
# Ubuntu/Debian
sudo apt-get install fonts-noto-cjk fonts-babelstone

# Arch Linux
sudo pacman -S noto-fonts-cjk
```

### 方案二：使用在线字体（无需安装）

项目已经配置了字体回退机制，会优先使用系统字体，如果系统没有则尝试加载在线字体。

### 方案三：使用备用显示方式

如果确实无法显示，可以考虑：
1. 使用拼音或描述代替
2. 使用图片展示
3. 使用相似字符代替

## 字根分类

### 已完全支持的字根区域

✅ **基本汉字** (U+4E00-U+9FFF) - 所有系统默认支持
✅ **CJK 扩展 A** (U+3400-U+4DBF) - 大部分系统支持
✅ **偏旁部首** (U+2F00-U+2FDF) - 大部分系统支持
✅ **康熙部首** (U+2F00-U+2FDF) - 大部分系统支持

### 需要特殊字体支持的字根区域

⚠️ **CJK 扩展 B** (U+20000-U+2A6DF) - 需要 BabelStone Han 等字体
⚠️ **CJK 扩展 C** (U+2A700-U+2B73F) - 需要 BabelStone Han 或 Hanazono
⚠️ **CJK 扩展 D** (U+2B740-U+2B81F) - 需要 BabelStone Han 或 Hanazono
⚠️ **CJK 扩展 E** (U+2B820-U+2CEAF) - 需要 BabelStone Han 或 Hanazono
⚠️ **CJK 扩展 F** (U+2CEB0-U+2EBEF) - 需要 BabelStone Han 或 Hanazono
⚠️ **CJK 扩展 G** (U+30000-U+3134F) - 需要最新 BabelStone Han
⚠️ **CJK 扩展 H** (U+31350-U+323AF) - 需要最新 BabelStone Han
⚠️ **私用区** (U+E000-U+F8FF) - 无标准字形

## 推荐字体优先级

### 完整支持（推荐）

1. **BabelStone Han** - 支持最全（强烈推荐）
   - 支持：扩展 B/C/D/E/F/G/H 区
   - 下载：https://www.babelstone.co.uk/Software/Download.html

2. **花园明朝体 (Hanazono)** - 免费开源
   - 支持：扩展 B/C/D/E/F 区
   - 下载：http://fonts.jp/hanazono/

3. **Windows 10/11 自带字体**
   - SimSun-ExtB (宋体扩展 B)
   - SimSun-ExtC (宋体扩展 C)
   - SimSun-ExtD (宋体扩展 D)
   - 位置：C:\Windows\Fonts\

### 部分支持

4. **Noto Sans CJK SC** - Google 官方
   - 支持：基本汉字 + 扩展 A/B 区

5. **Source Han Sans SC** - Adobe 官方
   - 支持：基本汉字 + 扩展 A/B 区

6. **Microsoft YaHei** - Windows 默认
   - 支持：基本汉字 + 扩展 A 区

## 常见问题

### Q: 为什么有些字根显示为方框？
A: 系统字体不支持该 Unicode 字符，需要安装扩展汉字字体。

### Q: 安装字体后仍然无法显示？
A: 请重启浏览器，清除缓存。

### Q: 移动端如何支持？
A: iOS 9+ 和 Android 5.0+ 默认支持大部分扩展汉字。
