## How to use

### 🔍 Quick Search
Type a radical character, hint, code, or category in the search box to locate results quickly.

### 🏷️ Category Filter
Use category tags to focus on a specific group:
- **All**: show every radical
- **Nature**: nature-related radicals
- **Human Body**: body-related radicals
- **Animals**: animal-related radicals
- **Plants**: plant-related radicals
- **Numbers**: number-related radicals
- **Strokes**: basic stroke radicals
- **Structure**: structural components
- **Others**: remaining radicals

### 🔤 Code Filter
Pick a specific code letter (A-Z) from the dropdown to narrow down the list.

### 📊 Sorting
- **By Code**: alphabetical order
- **By Character**: Unicode order

### 📄 Pagination
50 radicals are shown per page. Use pagination controls to continue browsing.

## Study Tips

1. Start with 20 core radicals
2. Memorize by category
3. Build code-to-radical associations
4. Reinforce through practice modes
