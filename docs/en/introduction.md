# Quick Start

Welcome to Tiger Origin! This is an easy-to-learn and powerful shape-based input method.

## 🎯 Why Choose Tiger Origin?

### Easy to Learn
- **Only 20 Core Radicals** - Compared to Wubi's 130+ radicals, greatly reducing the learning curve
- **Get Started in 30 Minutes** - Beginners can quickly master radical encoding
- **Intuitive Design** - Radical design is intuitive and easy to remember

### Efficient Input
- **Four-Code Fixed Length** - Maximum 4 letters input
- **Low Duplication Rate** - Smooth typing without frequent word selection
- **Scientific Character Decomposition** - Based on Chinese character origin and structure, following natural patterns

### Wide Application
- Daily office work, professional writing, programming development
- Full platform support for Windows and Android

## 🚀 Quick Start

### Step 1: Understand the Radicals

Tiger Origin is based on 20 core radicals distributed across keyboard keys:

```
Q W E R T Y U I O P
 A S D F G H J K L
  Z X C V B N M
```

Each key corresponds to 1-2 radicals, and radicals with similar shapes are arranged on nearby keys.

### Step 2: Learn the Encoding

Tiger Origin's encoding rules:

- **One-Code Characters**: Common characters with one-key input (e.g., 我, 人, 的)
- **Two-Code Characters**: Most common characters with two-key input
- **Three-Code Characters**: Most characters with three-key input
- **Four-Code Characters**: Complex characters with four-key input

## 📖 Learning Suggestions

### Beginner Path

1. **Day 1**: Complete sequential practice to familiarize with all radicals (about 30-60 minutes)
2. **Day 2-3**: Random practice to reinforce memory (20-30 minutes daily)
3. **Week 1-2**: Common radicals special practice
4. **Ongoing**: Use in daily life to form muscle memory

### Learning Tips

- **Understanding Memory**: Understand the relationship between radicals and Chinese characters, don't memorize mechanically
- **Multiple Repetitions**: Short and frequent sessions are more effective than long single sessions
- **Practical Application**: Use in real scenarios as soon as possible
- **Error Review**: Focus on practicing incorrect radicals

### Time Schedule

| Stage | Time | Goal |
|-------|------|------|
| Beginner | 1-3 days | Master 20 radicals |
| Practice | 1-2 weeks | Accuracy reaches 80% |
| Advanced | Ongoing | Speed reaches 60 WPM |
