## 🎯 需求 1：所有字根展示页面

### 功能规划

#### 1.1 页面结构设计

**页面位置**: `/roots/index.md` 或 `/all-roots.md`

**核心功能模块**:
```
┌─────────────────────────────────────┐
│   搜索栏 + 筛选器                    │
├─────────────────────────────────────┤
│   分类标签页 (全部 | 自然 | 人体 | ...) │
├─────────────────────────────────────┤
│   字根网格展示区                     │
│   [字根卡片] [字根卡片] [字根卡片]    │
│   [字根卡片] [字根卡片] [字根卡片]    │
├─────────────────────────────────────┤
│   分页/加载更多                      │
└─────────────────────────────────────┘
```

#### 1.2 字根卡片设计

每个字根卡片包含：
```vue
<RootCard 
  :character="白"      // 字根字符（大字号显示）
  :code="k"            // 编码（角标显示）
  :hint="白色"         // 提示（底部显示）
  :category="自然"     // 分类（标签显示）
  @click="showDetail"  // 点击显示详情
/>
```

#### 1.3 筛选功能

**筛选维度**:
- **按编码**: a-z 26 个字母筛选
- **按分类**: 自然、人体、动物、植物、数字、笔画、结构、其他
- **按难度**: 初级（核心 20 个）、中级（常用 100 个）、高级（全部）
- **搜索**: 支持字根字符、提示文字搜索

#### 1.4 排序功能

**排序选项**:
- 默认（编码顺序）
- 按使用频率
- 按学习时间（从易到难）
- 按笔画数

### 技术实现

#### 2.1 新建组件

**文件**: `docs/.vitepress/components/RootsGallery.vue`

```vue
<script setup>
import { ref, computed } from 'vue'
import { allRoots } from '../data/rootData.js'
import { rootsByCategory, getAllCategories } from '../data/rootDataEnhanced.js'
import RootCard from './RootCard.vue'

// 搜索关键词
const searchQuery = ref('')

// 选中的分类
const selectedCategory = ref('all')

// 选中的编码
const selectedCode = ref('')

// 排序方式
const sortBy = ref('code') // 'code' | 'frequency' | 'difficulty'

// 过滤后的字根列表
const filteredRoots = computed(() => {
  let result = [...allRoots]
  
  // 搜索过滤
  if (searchQuery.value) {
    const query = searchQuery.value.toLowerCase()
    result = result.filter(root => 
      root.character.includes(query) ||
      root.hint.toLowerCase().includes(query) ||
      root.code.toLowerCase().includes(query)
    )
  }
  
  // 分类过滤
  if (selectedCategory.value !== 'all') {
    result = result.filter(root => root.category === selectedCategory.value)
  }
  
  // 编码过滤
  if (selectedCode.value) {
    result = result.filter(root => root.code === selectedCode.value)
  }
  
  // 排序
  if (sortBy.value === 'code') {
    result.sort((a, b) => a.code.localeCompare(b.code))
  } else if (sortBy.value === 'frequency') {
    // 按频率排序逻辑
  }
  
  return result
})

// 统计信息
const totalCount = computed(() => filteredRoots.value.length)
</script>

<template>
  <div class="roots-gallery">
    <!-- 搜索和筛选区 -->
    <div class="filter-section">
      <input 
        v-model="searchQuery"
        type="search"
        placeholder="搜索字根..."
        class="search-input"
      />
      
      <!-- 分类标签 -->
      <div class="category-tabs">
        <button 
          v-for="cat in ['all', ...getAllCategories()]"
          :key="cat"
          :class="{ active: selectedCategory === cat }"
          @click="selectedCategory = cat"
        >
          {{ cat === 'all' ? '全部' : cat }}
        </button>
      </div>
      
      <!-- 编码筛选 -->
      <select v-model="selectedCode" class="code-select">
        <option value="">所有编码</option>
        <option v-for="code in 'abcdefghijklmnopqrstuvwxyz'" :key="code" :value="code">
          {{ code.toUpperCase() }}
        </option>
      </select>
      
      <!-- 排序选项 -->
      <select v-model="sortBy" class="sort-select">
        <option value="code">按编码排序</option>
        <option value="frequency">按频率排序</option>
        <option value="difficulty">按难度排序</option>
      </select>
    </div>
    
    <!-- 统计信息 -->
    <div class="stats">
      共 {{ totalCount }} 个字根
    </div>
    
    <!-- 字根网格 -->
    <div class="roots-grid">
      <RootCard
        v-for="root in filteredRoots"
        :key="`${root.character}-${root.code}`"
        :root="root"
      />
    </div>
    
    <!-- 加载更多/分页 -->
    <div class="pagination" v-if="totalCount > 50">
      <!-- 分页组件 -->
    </div>
  </div>
</template>
```
