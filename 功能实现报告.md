## 📊 项目概况

**项目名称**: 字源形码输入法学习系统

### 4. 工具函数库

#### 4.1 分组练习引擎（GroupPracticeEngine.js）
**文件**: [`GroupPracticeEngine.js`](file:///d:/vs_code/ziyuan2/docs/.vitepress/utils/GroupPracticeEngine.js)

**核心功能**:
```javascript
export function useGroupPractice(config = {}) {
  // 配置参数
  const cfg = {
    groupSize: 8,        // 每组字根数量
    repetitions: 4,      // 每字根练习遍数
    errorThreshold: 3,   // 错误阈值
  }
  
  // 返回的状态和方法
  return {
    // 状态
    currentRoot,        // 当前字根
    progress,          // 进度百分比
    accuracy,          // 正确率
    masteryRate,       // 掌握率
    masteredCount,     // 已掌握数量
    remainingCount,    // 剩余数量
    
    // 方法
    initPractice,      // 初始化练习
    validateInput,     // 验证输入
    handleNext,        // 下一个字根
    // ...
  }
}
```

**智能特性**:
- ✅ 自动分组（5-10 个字根/组）
- ✅ 组内随机循环
- ✅ 错误字根加入下一组
- ✅ 实时统计追踪
